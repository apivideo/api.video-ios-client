# Getting started

## Installation

### Carthage

Specify it in your `Cartfile`:

```
github "apivideo/api.video-ios-client" ~> 1.2.0
```

Run `carthage update`

### CocoaPods

Add `pod 'ApiVideoClient', '1.2.0'` in your `Podfile`

Run `pod install`
